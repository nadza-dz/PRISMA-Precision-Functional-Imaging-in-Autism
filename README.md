# PRISMA: A precision functional imaging dataset of autistic and non-autistic adults

This repository accompanies the PRISMA dataset published on OpenNeuro under accession number [ds007182](https://openneuro.org/datasets/ds007182/versions/1.0.0) and described in a preprint on [BioRxiv](https://doi.org/10.64898/2026.01.12.698952). It contains all scripts used for data preprocessing and multimodal quality control.
